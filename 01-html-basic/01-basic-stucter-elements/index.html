ส่วนหัวของเว็บไซต์ (Header)

นี่คือพื้นที่สำหรับโลโก้และเมนูนำทาง

นี่คือหัวข้อในกล่อง

คุณเห็นกล่องนี้แล้วใช่ไหม? มันถูกตกแต่งด้วย CSS ครับ

ชื่อคลาส "mycontainer" ทำให้เราสามารถเลือกตกแต่งกล่องนี้โดยเฉพาะได้

นี่คือข้อความที่อยู่นอกกล่อง.



โน้ต: โค้ดด้านล่างนี้คือตัวอย่าง Emmet สำหรับสร้าง Sibling Elements:

header+nav+main+footer

ซึ่งจะขยายเป็นโค้ด HTML ดังนี้:

<header></header>
<nav></nav>
<main></main>
<footer></footer>

Emmet แบบนี้มีไว้เพื่อลดขั้นตอนในการเขียนโค้ดและช่วยให้เห็นโครงสร้างหลักของหน้าเว็บได้รวดเร็ว


ข้อความจำลอง 1 ย่อหน้า กับ 3 ย่อหน้า
