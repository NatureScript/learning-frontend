Example 2: Generating Placeholder Text (Lorem Ipsum)

To quickly insert placeholder text, you can use `lorem` followed by the number of words. For paragraphs, `p*N>lorem` can generate multiple paragraphs with random text:

  • `p>lorem` for one paragraph of random text.
  • `p*3>lorem` for three paragraphs of random text.

This is a quick way to populate your layout with sample content.
