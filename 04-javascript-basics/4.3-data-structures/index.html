ผลลัพธ์จาก Console

ผลลัพธ์ส่วนใหญ่จากบทเรียนนี้จะแสดงใน **"Console"** ของ Developer Tools. โปรดเปิด Developer Tools (กด F12) เพื่อดูผลลัพธ์ของโค้ด JavaScript เกี่ยวกับ Arrays และ Objects.

ส่วนนี้เป็นเพียงพื้นที่แสดงผลเพิ่มเติม หากมีการใช้ DOM Manipulation ในอนาคต.
